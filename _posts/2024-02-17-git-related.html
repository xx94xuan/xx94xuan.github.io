Update email for existing commits

make sure you update the desired email at this time:
git config user.email
update the email for the latest commit:
git commit --amend --reset-author
update the email for previous commits:
git rebase -i HEAD~10
choose `e`/`edit` to the targeted commits, then run git commit --amend --reset-author,
git rebase --continue use `-f` to force push the new branch to the origin.
